## Authentication APIs

### Register User

* **POST** /api/auth/register
* Registers a new user and sends an
  email OTP verification code.

### Verify Registration OTP

* **POST** /api/auth/verify-otp
* Verifies the user's email using the OTP.

### Resend OTP

* **POST** /api/auth/resend-otp
* Sends a new OTP to users who have not
  verified their email.

### Forgot Password

* **POST** /api/auth/forgot-password
* Sends a password reset email.

### Reset Password

* **POST** /api/auth/reset-password
* Resets the user's password using the
  reset token.

### Login

* **POST** /api/auth/login
* Authenticates a verified user and
  returns a Laravel Sanctum Bearer token.

### Get Current User

* **GET** /api/auth/me
* Returns the authenticated user's
  profile information.

### Logout

* **POST** /api/auth/logout
* Revokes the current access token.

### Logout All Devices

* **POST** /api/auth/logout-all
* Revokes all active access tokens.

## User Management APIs

### Get All Users

* **GET** /api/admin/users
* Returns all registered users.

### Activate User

* **PATCH** /api/admin/users/{id}/activate
* Activates a user account.

### Deactivate User

* **PATCH** /api/admin/users/{id}/deactivate
* Deactivates a user account and prevents
  future logins.

## Role-Based Access Control

### Admin Route

* **GET** /api/admin/test
* Accessible only by Admin users.

### Support Agent Route

* **GET** /api/agent/test
* Accessible only by Support Agent users.

### User Route

* **GET** /api/user/test
* Accessible only by User accounts.
