# IT Help Desk & Ticketing Management System

## Project Overview

The IT Help Desk & Ticketing Management System is a full-stack web application designed to streamline technical support operations within an organization. Employees can submit support requests, while IT support agents and administrators can manage, prioritize, assign, and resolve tickets through a centralized dashboard.

This project is being developed as part of a **Full Stack Web Development Internship**.

---

## Project Objectives

- Understand the project scope
- Design professional UI wireframes
- Create a relational database schema
- Prepare ERD and workflow diagrams
- Plan the application architecture
- Design RESTful APIs
- Follow collaborative development using GitHub

## System Users

| Role | Description |
|------|-------------|
| Admin | Full system access |
| Employee | Creates and tracks support tickets |
| IT Support Agent | Manages and resolves tickets |
| Manager | Monitors tickets and reports |
